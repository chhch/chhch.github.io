## Übersicht

-   `npm init`
-   `npm install [-g]`
    -   Lokale Installation: Dateien werden in `./node_modules` des aktuellen Ordners oder eines darüber liegenden Ordners abgelegt
    -   Globale Installation: Dateien werden in `/usr/local` abgelegt.
-   `npm uninstall [-g]`
-   `npm list [-g] --depth=0`
-   `npm outdated [-g]`
-   `npm update [-g]`
-   `npm doctor`
